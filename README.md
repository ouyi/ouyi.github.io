# Memory Spills

Memory Spills is [a blog based on Jeykll and hosted on GitHub](https://ouyi.github.io).

## Useful commands

Test locally:

    bundle exec jekyll serve --port 8000 --host 0.0.0.0 --drafts

Show a bundle package content:

    bundle show minima
